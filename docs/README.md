## Quick start

- 1) Generate the API key from Mobilpay Admin
- 2) Set the "API Key" as header of your request
       Header of Request
          "Authorization" => "YOUR ACTUAL API KEY FROM MOBILPAY"
          "Content-Type"  => "application/json"
- 3) Send the request by POST Method to "https://secure.sandbox.netopia-payments.com/"
